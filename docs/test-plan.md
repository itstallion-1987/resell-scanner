# Тест-план: 30 наборов фото (шаг 1 порядка работы)

Прогон: `node proxy/test/run-test-set.mjs http://localhost:8787 <platform>`.
Каждый кейс — папка `proxy/test/sets/<id>-<имя>/` с 1–3 фото.

## Состав набора

| # | Категория | Кейсов | Что проверяем |
|---|---|---|---|
| 1–8 | Одежда **с биркой** (джинсы, куртка, платье, футболка брендовая, свитер, детское, спорт, пальто) | 8 | бренд/размер/состав с бирки, точность ≥ 85% |
| 9–12 | Одежда **без бирки** | 4 | brand=null + retry_hint «сфотографируйте бирку», confidence=low/medium |
| 13–16 | Обувь (кроссовки брендовые, ботинки с потёртостями, туфли, детская) | 4 | модель по дизайну, дефекты в condition_details |
| 17–21 | **Электроника** (наушники, консоль/геймпад, фотоаппарат, колонка, клавиатура) | 5 | категория/модель, не-одежда обязательна (клин против ThreadMint) |
| 22–24 | **Предметы быта** (посуда, лампа, игрушка) | 3 | категория, разумная вилка цены |
| 25–26 | Вещи с явными дефектами (пятно на одежде, скол на посуде) | 2 | дефект упомянут в condition_details в 100% случаев |
| 27 | Брендовая сумка | 1 | НЕТ вердикта подлинности; «tag reads X» формулировки |
| 28 | Ювелирка/бижутерия | 1 | price_range.low/high = null, отказ от оценки по фото |
| 29 | Плохой свет / размытое фото | 1 | recognized=false + конкретный retry_hint |
| 30 | «Не товар» (стол, стена, кот) | 1 | recognized=false |

## Чек-лист по каждому кейсу

- [ ] Latency от запроса до ответа ≤ 5 с (поле `seconds` в results.json)
- [ ] `title` в лимите платформы, ключи спереди
- [ ] Видимые дефекты → `condition_details`; невидимых выдумок нет
- [ ] Нет слов authentic/genuine/fake/replica и аналогов
- [ ] `price_range.note` содержит дисклеймер; `sold_comps_query` вставляется в поиск платформы как есть
- [ ] Себестоимость по usage ≤ $0.04
- [ ] Субъективно: «вставил и опубликовал без правок» — да/нет (цель ≥ 70% по набору)

## Прогон по платформам

Полный набор гонять на `ebay`; кейсы 1–4 дополнительно на `vinted`, `poshmark`, `depop` — проверить тон, лимит заголовка, хэштеги (vinted/depop), буллеты (poshmark).
